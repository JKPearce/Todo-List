# Todo-List
Todo list to allow users to keep track of tasks they need to do

Live Demo: [Here](https://jkpearce.github.io/Todo-List/)

## Goals
The goal in this project is to understand ES6 modules by using npm and webpack.

### Tech used 
- HTML
- SCSS
- JavaScript
- NPM
- WebPack
